TradingAssistant

I created this project largely because I found myself continuously checking the prices of certain assets, and waiting for them to reach a certain price. Since the assets I was looking at are not mainstream and there is no real price alert system for them I decided to create one. Using Python 3, the Python telegram-bot library, SQLite, and various public exchange API's the TradingAssistant was born.

This bot allows users to set price alerts and fetch the current price of any stock or cryptocurrency on the following exchanges:

  • NASDAQ
  • NYSE
  • Binance
  • BitMex
  • Coinbase Pro
  • Bittrex
  • Qtrade

Users will soon have the option of receiving alerts based on technical indicators, large price movements, and sentiment analysis!
