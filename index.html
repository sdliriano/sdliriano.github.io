MKTS

A web application that lists stocks, foreign exchange currencies, and cryptocurrencies ordered by market cap.


  • Created using the LAMP technology stack
  • Database created and managed with SQL, MySQL, and PHPMyAdmin
  • Hosted on an Amazon Web Services CentOS server

TradingAssistant

Telegram Bot that can set price alerts and fetch the current price of any given stock, foreign exchange currency & cryptocurrency. Price alerts can be based on price movements and technical indicators.


  • Written in Python 3
  • User data stored using SQLite database
  • Asset data retrieved via an exchange API module I created.


EthPolls

First Decentralized Poll Voting System Held on the Ethereum Blockchain

  • DApp Frontend built with ReactJS, Create-React-App & CSS
  • Smart Contracts written in Solidity
  • Deployed using Surge
  • Sybil Resistant


RSI Backtester

Software that tests various RSI trading strategies with price history of digital assets. Results are stored in a SQLite DB file.


  • Created using Python 3
  • Asset and indicator data fetched using two separate API's
  • Utilizes SQLite to store resulting data.


CryptoMatch

Crypto Vote & Crypto Match are tools that can be used to help determine the sentiment in the market and to find a coin that matches your standards.


  • Written in HTML5, CSS, JavaScript & PHP
  • Utilizes read and write to files for basic data storage
  • Asset data retrieved via exchange API calls
  • Over 3,000 total visitors


ARC

This is a React/Flask (Python) blockchain based project to teach estimation & finance to all ages. Created during the PhillyCodefest Hackathon


  • Created Frontend using ReactJS
  • Utilizes Hyperledger Sawtooth Blockchain
  • Team members utilized Git Version Control
